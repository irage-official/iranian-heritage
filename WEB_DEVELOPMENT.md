# راهنمای توسعه وب - Web Development Guide

## 🚀 اجرای پروژه با پورت ثابت

### روش ۱: استفاده از اسکریپت (پیشنهادی)
```bash
./run_web.sh
```

### روش ۲: دستور مستقیم Flutter
```bash
flutter run -d chrome --web-port=8080 --web-hostname=localhost --hot
```

### روش ۳: استفاده از VS Code
1. فایل `.vscode/launch.json` را باز کنید
2. روی "Flutter Web (Fixed Port)" کلیک کنید
3. یا از کلید میانبر `F5` استفاده کنید

## 🔄 Hot Reload

پس از اجرای پروژه:
- ✅ تغییرات کد به صورت خودکار اعمال می‌شود
- ✅ نیازی به بستن مرورگر نیست
- ✅ فقط صفحه را رفرش کنید (F5 یا Ctrl+R)
- ✅ پورت همیشه 8080 خواهد بود

## 🌐 آدرس پروژه

```
http://localhost:8080
```

## 🛠 تنظیمات اضافی

### تغییر پورت (در صورت نیاز)
در فایل `run_web.sh` یا دستور Flutter، پورت را تغییر دهید:
```bash
--web-port=3000  # برای پورت 3000
```

### فعال‌سازی Debug Mode
```bash
flutter run -d chrome --web-port=8080 --debug --verbose
```

## 📝 نکات مهم

1. **پورت ثابت**: همیشه از پورت 8080 استفاده می‌شود
2. **Hot Reload**: تغییرات کد به صورت خودکار اعمال می‌شود
3. **مرورگر**: نیازی به بستن و باز کردن مجدد نیست
4. **رفرش**: فقط صفحه را رفرش کنید تا تغییرات را ببینید

## 🔧 عیب‌یابی

### اگر پورت در حال استفاده است:
```bash
lsof -ti:8080 | xargs kill -9
```

### اگر پروژه اجرا نمی‌شود:
```bash
flutter clean
flutter pub get
flutter run -d chrome --web-port=8080
```
